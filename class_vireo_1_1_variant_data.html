bool Vireo::VariantData::CopyToNonVariant ( TypeRef  destType,
void *  destData 
) const

Copy only data from a variant (even if nested) to a destination whose type is non-variant – which also means there is no need (no way) to copy attributes at any nested level. If void-variant is what the innermost we find, then there is no real data associated with this variant of non-variant type, so return false. if found and copied, return true.
